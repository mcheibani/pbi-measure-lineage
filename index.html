        Load Measure Lineage Data

        Load lineage via DAX Query View, CSV export, or VPAX archive.

        1Run this query in DAX Query View
        Open your model in Power BI Desktop, use DAX Query View, and run this query to return measure dependencies only.
        EVALUATE VAR _deps = INFO.CALCDEPENDENCY() VAR _filtered = FILTER( _deps, [OBJECT_TYPE] = "MEASURE" && NOT CONTAINSSTRING([TABLE], "LocalDateTable_") && NOT CONTAINSSTRING([TABLE], "DateTableTemplate_") && NOT CONTAINSSTRING([REFERENCED_TABLE], "LocalDateTable_") && NOT CONTAINSSTRING([REFERENCED_TABLE], "DateTableTemplate_") ) RETURN SELECTCOLUMNS( _filtered, "FromObject", [OBJECT], "FromTable", [TABLE], "ToObject", [REFERENCED_OBJECT], "ToTable", [REFERENCED_TABLE], "DependencyType", [REFERENCED_OBJECT_TYPE] )
        2Paste results below
        Accepts tab-separated DAX Query View output directly.
        Expected columns: FromObject, FromTable, ToObject, ToTable, DependencyType
        DependencyType is optional — defaults to "Measure" if missing.
